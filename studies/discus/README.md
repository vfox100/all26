# Discus

A simple rotary demo, with nontrivial mass, to teach controls:

* Measurement
* Current limits
* Feedforward
* Feedback
* Profiles

A good controller is the "knobs" control
or the "midi" control, both from 2024.

This uses the same pattern as the Fivebar, with various
"setups" that demonstrate different levels of control
sophistication, using different Subsystem implementations.
