# Vscode settings

Install these extensions:

* mypy type checker (microsoft)
* pylint (microsoft)
* black formatter (microsoft)

## Windows

Windows seems to come with python 3.11, which is antique by now.

To install 3.14, go to https://www.python.org/downloads/.

You can also uninstall 3.11 using the windows programs adder control panel.

you'll need to install numpy from the terminal:

* python -m pip install numpy
* python -m pip install typing_extensions
* python -m pip install opencv-python
* python -m pip install robotpy
* python -m pip install robotpy-apriltag
* python -m pip install robotpy-cscore
